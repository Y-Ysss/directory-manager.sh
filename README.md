# Directory Manager

## Description
Directory Bookmark Manager for easy access to directories on the bash.

## Getting Started
Copy the `script/yaml.sh` file and add alias to `.bash_aliases` file.

```sh
alias dirmages='{in each environment}/dirmages.sh'
```
